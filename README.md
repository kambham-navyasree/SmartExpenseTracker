# Smart Expense Tracker

**Live Demo:** https://kambham-navyasree.github.io/SmartExpenseTracker/

**How to Use:**
1. Open the link in any browser.
2. Fill in expense details (Amount, Category, Date, Description).
3. Click "Add Expense".
4. See your expense added in the list and total updated.
5. To remove an expense, click the delete button next to it.
