# Smart Expense Tracker

**Live Demo:** [https://kambham-navyasree.github.io/SmartExpenseTracker/](https://kambham-navyasree.github.io/SmartExpenseTracker/)

## Features
- Add, delete, and track daily expenses
- See total expenses updated in real time
- Connected frontend (GitHub Pages) → backend (Render API)
- Fully functional and live

## How to Use
1. Open the live demo link in any browser.
2. Enter expense details: Amount, Category, Date, Description.
3. Click **Add Expense** to add the expense.
4. Click **Delete** to remove an expense.
5. Total expense updates automatically.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (hosted on Render)
- Deployment: GitHub Pages
